# 📚 BookHeaven

**BookHeaven** is a beautifully crafted online bookstore web application designed for book lovers and reading enthusiasts. With a clean UI, easy navigation, and both user and admin functionalities, it offers a seamless digital bookstore experience.


## ✨ Overview

BookHeaven allows users to:

- 🏠 Explore a curated homepage with featured books.
- 📖 Browse a full catalog of books with cover images, authors, and prices.
- 🔍 View detailed information about each book on a dedicated page.
- 🛒 Add books to the cart (with quantity control and price summary).
- 🔐 Create an account or login as an existing user.
- 👤 View user-specific profile and cart data.
- 🧑‍💼 Admins can log in to manage the book inventory via a separate panel.


## 🧩 Key Highlights

- 🎨 **Responsive UI** built with **Tailwind CSS**.
- 💬 Clean navigation bar with dynamic links based on login role (`user` or `admin`).
- 🖼️ **Fallback image** support — displays `tom-and-jerry.jpeg` for books without a valid image URL.
- 📂 Book cards are visually appealing with hover animations.
- 🔐 Role-based routing (admin vs user) and access control.
- 📦 Optimized performance and hosted on **Netlify** for fast loading.

## ⚙️ Tech Stack

- **Frontend**: React, React Router DOM, Redux Toolkit, Tailwind CSS
- **Backend**: Node.js, Express.js, MongoDB
